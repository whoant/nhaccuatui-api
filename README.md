# Nhaccuatui API

NhaccuatuiAPI is a NodeJS library.

## Installation

```bash
npm i nhaccuatui-api
```

## Usage

### 1.Get home

```javascript
Nhaccuatui.getHome();
```

### 2. Get info music

Example: https://beta.nhaccuatui.com/bai-hat/laylalay-jack-j97.n8VupC0HXKgY.html

```javascript
Nhaccuatui.getSong('n8VupC0HXKgY');
```

### 3. Get details playlist or album

Example: https://beta.nhaccuatui.com/playlist/future-k-pop-2021-va.K6KRu3xHK9MN.html

```javascript
Nhaccuatui.getPlaylist('K6KRu3xHK9MN');
```

### 4: Get Lyric

Example: https://beta.nhaccuatui.com/bai-hat/laylalay-jack-j97.n8VupC0HXKgY.html

```javascript
Nhaccuatui.getLyric('n8VupC0HXKgY');
```

### 5: Get top 20

```javascript
Nhaccuatui.getTop20();
```

### 6: Get top 100

Example: https://beta.nhaccuatui.com/top100/top-100-nhac-tre.m3liaiy6vVsF.html

```javascript
Nhaccuatui.getTop100('m3liaiy6vVsF');
```

### 7: Get topic

```javascript
Nhaccuatui.getTopic();
```
